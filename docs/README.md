## 目录结构

- `product/` — 产品文档与里程碑（PRD、里程碑计划）。
- `architecture/` — 技术架构、模块设计、前端规范与集成说明。
- `ops/` — 部署指南、配置说明、演示脚本、问题追踪、交接进度。
- `research/` — 市场/协议调研材料。
- `archive/` — 旧版讨论与长篇对话记录，仅作参考。

如需新增文档，请按类别存放，并在本索引更新链接，确保团队成员与评委可快速定位信息。
